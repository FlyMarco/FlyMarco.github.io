View 动画

Tween 动画

Tween动画主包括 RotateAnimation (旋转)、 ScaleAnimation (缩放)、 TranslateAnimation (平移)、 AlphaAnimation (透明度)、 AnimationSet (动画集合)。

  1. 代码创建

    1. 单个动画
// 0，180 为起始值和终止值, float 类型
AlphaAnimation alphaAnimation = new AlphaAnimation(0，180);

// 配置参数
alphaAnimation.setXXX();

// 为一个 view 开启动画
view.startAnimation(alphaAnimation);

2. 动画集合
// false 各个 animation 使用自己的 Interpolator，true 共用 set 的 Interpolator
AnimationSet animationSet = new AnimationSet(false);

// 添加 Animation
animationSet.addAnimaion(XXX);

// 配置参数
animationSet.setXXX();

// 启动动画
view.startAnimation(animationSet);

  2. XML 创建

    1. xml 文件, /res/anim/anim_sample.xml
// 单个动画
<alpha></alpha>

// 动画集合
<set>
    <alpha/>
    <translate/>
</set>

2. java 加载
Animation animation = AnimationUtils.loadAnimation(context, R.anima.anim_sample);

属性动画

改变对象的属性

Duration    // 指定动画持续时间，默认时间是300ms
TimeInterpolation    // 时间插值器，如加速、加速等。
Repeat count and behavior    // 重复次数
Animation Set    // 动画合集。用来同时或者顺序播放多个动画。
Frame Refresh Delay    // 多长时间刷新一次，默认是10ms。
TypeEvaluator    // 类型估值，主要用于设置动画操作属性的值。
ObjectAnimator

ObjectAnimator 要求对象有 setter 和 getter 方法，并且对属性的修改能够通过某种方式反映。

/×
 × 创建 ObjectAnimator，ofFloat、ofInt、ofObject 等
 × 只有一个参数表示变化的终点值，会通过 getter 方法获取当前值为起始值
 × 两个参数表示变化的起始值、终点值
 × 多个参数中包含起始值、中间值、终点值
 ×/
ObjectAnimator objectAnimator = ObjectAnimator.ofFloat(view, "translationX", 0, 180F);

// 设置参数，如 Duration、TimeInterpolator 等
objectAnimator.setXXX();

// 开启动画
objectAnimator.start();
ValueAnimator

ValueAnimator 计算属性值，根据属性值改变对象的属性

// 创建 ValueAnimator，通过 ofFloat、ofInt、ofObject 等
ValueAnimator valueAnimator = ValueAnimator.ofFloat(30F, 180F, 60F);

// 设置时间插值器、类型估值器等
valueAnimator.setXXX();

// 设置监听，改变对象属性
valueAnimator.addUpdateListener(new AnimatorUpdateListener() {        
    @Override
    public void onAnimationUpdate(ValueAnimator animation) {
        // TODO Auto-generated method stub            
    }
});

// 开启动画
valueAnimator.start();

// AnimatorSet
playSequentially(...);    // 顺序执行

play(X)

//启动动画
animatorSet.start();

PropertyValuesHolder

保存动画期间属性的变换值，可以通过 ObjectAnimator 和 ValueAnimator 创建动画

// 创建 PropertyValuesHolder
PropertyValuesHolder rotateX = PropertyValuesHolder.ofFloat("rotateX", 0, 180);
PropertyValuesHolder rotateY = PropertyValuesHolder.ofFloat("rotateY", 0, 180);

// 创建 Animator
ValueAnimator o = ValueAnimator.ofPropertyValuesHolder(rotateX, rotateY);
ObjectAnimator oa = ObjectAnimator.ofPropertyValuesHolder(view, rotateX, rotateY);

SVG 动画

Android 5.X 提供 VerctorDrawable 和 AnimatedVectorDrawable 支持SVG

  1. SVG 简介

    // xml 标签示例
<vector>
    <group>
        <path/>
    </group>
    <path/>
</vector>

// SVG 常用指令
M = moveto(M X,Y)    // 移动到指定坐标
L = lineto(L X,Y)    // 画直线到指定位置
H = horizontal lineto(H X)    // 画横线
V = Vertical lineto(V Y)    // 画竖线
C = curveto(C X1,Y1,X2,Y2,X,Y)    // 三次贝塞尔曲线
S = smooth curveto(S X2,Y2,X,Y)    // 三次贝塞尔曲线
Q = quadratic Belzier curve(Q X1,Y1,X,Y)    // 二次贝塞尔曲线
T = smooth quadratic Belzier curve(T X,Y)    // 映射前面路径的终点
A = elliptical Arc(A RX,RY,XROTATION,FLAG1,FLAG2,X,Y)    // 弧线
Z = closepath()    // 闭合路径

1. X,Y 为重点坐标，指令大写为绝对坐标，小写为相对坐标
2. 同一指令出现多次可使用一个

Transitions between Activities

  1. Fade、Slide、Explode

    // xml
<?xml version="1.0" encoding="utf-8"?>
<slide xmlns:android="http://schemas.android.com/apk/res/"
    android:duration="1000"/>
Fade fade = TransitionInflater.from(context).inflateTransition(R.transition.fade);;

// Java
Fade fade = new Fade();
fade.setDuration(1000);

// 使用
getWindow().setEnterTransition(fade);
getWindow().setExitTransition(fade);
getWindow().setReturnTransition(fade);
getWindow().setReenterTransition(fade);

// 是否覆盖前一个动画
getWindow().setAllowReturnTransitionOverlap(false);
getWindow().setAllowEnterTransitionOverlap(false);
