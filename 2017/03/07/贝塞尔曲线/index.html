贝塞尔曲线点坐标

每两个控制点确定下一层的一个控制点，直到最后一个点，就是曲线上的一个点

float[] points;  // length = n
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n - i; j++) {
        points[j] = (1 - t) * points[j] + t * points[j + 1];
    }
}
// points[0] 为最终求得的点
