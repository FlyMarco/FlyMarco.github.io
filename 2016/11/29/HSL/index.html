HSL简介

HSL 色彩模式是工业界的一种颜色标准，它通过对色相(H)、饱和度(S)、明度(L)三个颜色通道的变化，以及它们相互之间的叠加来得到各式各样的颜色的。

  • 色相：色彩的首要特征，用以区别不同的色彩，取值范围为[0,360)的圆心角，0红、60黄、120绿、180青、240蓝、300洋红
  • 饱和度：色彩的鲜艳程度，也称色彩的纯度，取决于含色成分和消色成分的比例，取值范围[0%,100%]，数值越大，灰色含量越少，色彩越鲜艳
  • 明度：颜色的明暗程度，白色明度最高，黑色明度最低，取值范围[0%,100%],数值越大，色彩越暗，越接近于黑色

任务要求

获得任意一张24位彩图图片，通过拖动条改变图片色彩饱和度，实现实时预览改变效果。主要过程包括：提取图片RGB的数据、RGB转到HSL格式、改变S的值、HSL到RGB数据格式转换，生成Bitmap进行预览。

  2. RGB->HSL格式转换

    首先计算RGB分量的max和min的值，然后获得HSL各分量的值

    for (int i = 0; i < rgb.length; i += 3) {
    max = Math.max(r, Math.max(g, b));
    min = Math.min(r, Math.min(g, b));
    h[j] = new BigDecimal(getH(r, g, b, max, min)).setScale(3, BigDecimal.ROUND_HALF_UP).doubleValue();
    s[j] = new BigDecimal(getS(max, min)).setScale(3, BigDecimal.ROUND_HALF_UP).doubleValue();
    l[j] = new BigDecimal(getL(max, min)).setScale(3, BigDecimal.ROUND_HALF_UP).doubleValue();
    j++;
}

    计算H分量的值，注意最后的取值范围

    private double getH(double r, double g, double b, double max, double min) {
double k = 0;
if (max == min) k = 0;
else if (max == r) k = ((g - b) / (max - min) ) % 6;
else if (max == g) k = (b - r) / (max - min) + 2;
else if (max == b) k = (r - g) / (max - min) + 4;
return (k * 60 < 0) ? k * 60 + 360 : k * 60 > 360 ? 360 : k * 60;
}

    计算L分量的值，注意最后的取值范围

    private double getL(double max, double min) {
double key = (max + min) / 2;
return key < 0 ? 0 : key > 1 ? 1 : key;
}

    计算S分量的值，注意最后的取值范围

    private double getS(double max, double min) {
double l = getL(max, min);
double key = (max == min) ? 0 : (max - min) / (1 - Math.abs(2 * l - 1));
return key < 0 ? 0 : key > 1 ? 1 : key;
}

  3. 改变S的值

    每一次计算S的值都是在原S数组clone的新数组，保证S原数组保持不变

    double[] s = S.clone();
for (int i = 0; i < S.length; i++) {
    s[i] = s[i] + s;
    s[i] = s[i] > 1 ? 1 : s[i] < 0 ? 0 : s[i];
}

  4. HSL->RGB格式转换

    计算得到c、h、x、m的值，通过中间值计算RGB的各分量的值

    byte[] d = new byte[H.length * 3];
int kk = 0;
for (int i = 0; i < H.length; i++){
    double c = (1 - Math.abs(2 * L[i] - 1)) * s[i];
    double h = H[i] / 60;
    double x = c * (1 - Math.abs(h % 2 - 1));
    double m = L[i] - c / 2;
    d[kk++] = (byte) getB(c, h, m, x);
    d[kk++] = (byte) getG(c, h, m, x);
    d[kk++] = (byte) getR(c, h, m, x);
}

    计算R分量的值，注意最后的取值范围

    private int getR(double c, double h, double m, double x) {
    double r = 0;
    if (h == 0) r = 0;
    if (h >= 0 && h < 1 || h >= 5 && h < 6) r = c;
    else if (h >= 1 && h < 2 || h >= 4 && h < 5) r = x;
    else if (h >= 2 && h < 4) r = 0;
    r = r + m;
    r = r >1 ? 1 : r < 0 ? 0 : r;
    return (int) (r * 255);
}

    计算G分量的值，注意最后的取值范围

    private int getG(double c, double h, double m, double x) {
    double g = 0;
    if (h == 0) g = 0;
    if (h >= 4 && h < 6) g = 0;
    else if (h >= 0 && h < 1 || h >= 3 && h < 4) g = x;
    else if (h >= 1 && h < 3) g = c;
    g = g + m;
    g = g >1 ? 1 : g < 0 ? 0 : g;
    return (int) (g * 255);
}

    计算B分量的值，注意最后的取值范围

    private int getB(double c, double h, double m, double x) {
    double b = 0;
    if (h == 0) b = 0;
    if (h >= 3 && h < 5) b = c;
    else if (h >= 2 && h < 3 || h >= 5 && h < 6) b = x;
    else if (h >= 0 && h < 2) b = 0;
    b = b + m;
    b = b >1 ? 1 : b < 0 ? 0 : b;
    return (int) (b * 255);
}
